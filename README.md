##TerraVolt
Designing Tomorrow's Communities

## Overview
TerraVolt is a community-level microgrid planning and sustainability assessment tool developed using Python and MATLAB.
The project helps estimate future electricity demand, analyze battery reserves, simulate IoT nodes, and provide sustainability recommendations for villages, hostels, campuses, and societies.

## Features
-Community Energy Planner

-Demand Growth Forecast

-Sustainability Score Engine

-Recommendation System

-Digital Twin Simulation

-Weather and Temperature Nodes

-Battery Reserve Analysis

-MATLAB-Based Analytics
